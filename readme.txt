Integración de Wompi para WooCommerce disponible en Panamá.

== Descripción ==

Payment Wompi WooCommerce for Panamá es un plugin que permite integrar la pasarela de pagos Wompi en WooCommerce para Panamá. Con esta herramienta, puedes aceptar pagos en línea de manera segura y eficiente.

== Instalación ==
1. Descarga el archivo [payment-wompi-woo-pa.zip](https://github.com/saulmoralespa/payment-wompi-woo-pa/releases/latest/download/payment-wompi-woo-pa.zip).
2. En tu panel de administración de WordPress, ve a **Plugins > Añadir nuevo**.
3. Haz clic en **Subir plugin** y selecciona el archivo `payment-wompi-woo-pa.zip`.
4. Haz clic en **Instalar ahora** y luego en **Activar**.
5. Configura el plugin en Woocommerce -> ajustes -> Pagos -> Wompi.
